# TextSniper for Windows - 요구사항 명세서

## 개요

Windows에서 화면의 특정 영역을 캡처하여 텍스트를 OCR로 인식하고 클립보드에 복사하는 경량 데스크톱 앱

## 시스템 요구사항

| 항목 | 요구사항 |
|------|----------|
| 최소 OS | Windows 10 버전 1903 (빌드 18362) 이상 |
| 런타임 | WebView2 (Windows 11 기본 포함, Windows 10은 설치 필요) |
| 언어팩 | 한국어 OCR 사용 시 Windows 한국어 언어팩 필요 |

## 핵심 흐름

1. 시스템 트레이에 상주
2. `Shift+Alt+T` 누르면 마우스 커서가 있는 모니터에 오버레이 표시
3. 마우스 드래그로 영역 선택 (십자선 커서)
4. 오버레이 즉시 닫힘 + 트레이 아이콘 처리 중 표시
5. Windows OCR API로 텍스트 인식 (영어 → 한국어 순차 시도)
6. 인식된 텍스트를 클립보드에 복사
7. 트레이 아이콘 완료 표시 (1초 후 원래 아이콘으로 복귀)

## 기술 스택

| 항목 | 선택 | 비고 |
|------|------|------|
| 프레임워크 | Rust + Tauri v2 | 트레이, 단축키, OCR 호출 담당 |
| 오버레이 | Win32 API 직접 구현 | `CreateWindowEx` + `WS_EX_LAYERED`. Tauri WebView 미사용 |
| 화면 캡처 | `BitBlt` (GDI) | 단순하고 안정적. UIPI 제한은 Known Limitation |
| OCR 엔진 | Windows OCR API | `windows` crate (`Windows.Media.Ocr`) |
| 클립보드 | `arboard` crate | 크로스플랫폼 클립보드 라이브러리 |
| 인식 언어 | 한국어 + 영어 | Windows 언어팩 기반 |

## 기능 요구사항

### 캡처 방식

- 마우스 드래그로 원하는 영역을 선택
- 십자선(crosshair) 커서 사용
- 반투명 오버레이로 화면을 덮고, 선택 영역 내부는 원본 밝기 유지
- 역방향 드래그 지원 (어느 방향이든 선택 가능)
- 최소 선택 영역: 10x10 픽셀 미만은 무시 (캡처 취소 처리)
- ESC 키로 캡처 취소
- 오버레이 표시 범위: 마우스 커서가 위치한 모니터 1개

### DPI 처리

- 앱은 Per-Monitor DPI Aware V2로 동작
- 오버레이 좌표는 물리 픽셀 기준으로 변환하여 캡처에 사용
- 캡처 비트맵은 물리 픽셀 기준으로 생성

### 시스템 트레이

- 앱 실행 시 시스템 트레이에 아이콘 표시
- 트레이 아이콘 우클릭 시 컨텍스트 메뉴:
  - 캡처 실행
  - 자동 실행 켜기/끄기
  - 종료
- 앱 중복 실행 방지 (단일 인스턴스)
- 탐색기 재시작 시 트레이 아이콘 자동 재등록 (`TaskbarCreated` 메시지 감지)

### 단축키

- `Shift+Alt+T`: 캡처 모드 진입 (글로벌 단축키)
- 등록 실패 시 트레이 아이콘 툴팁에 "단축키 등록 실패" 표시

### OCR 인식

- Windows OCR API (`Windows.Media.Ocr`) 사용
- 인식 순서: 영어 엔진 시도 → 결과가 비어있으면 한국어 엔진 시도
- 캡처한 HBITMAP → `SoftwareBitmap` 변환 후 OCR 수행
- OCR 처리 중 트레이 아이콘을 처리 중 아이콘으로 변경
- 처리 타임아웃: 5초

### 사용자 피드백 (토스트 대신)

- **처리 중**: 트레이 아이콘을 스피너/로딩 아이콘으로 변경
- **성공**: 트레이 아이콘을 체크 아이콘으로 1초간 표시 후 원래 아이콘 복귀
- **실패/빈 결과**: 트레이 아이콘을 X 아이콘으로 1초간 표시 후 원래 아이콘 복귀
- 텍스트가 없는 영역 캡처 시 클립보드에 쓰지 않음 (기존 클립보드 내용 보존)

### 자동 실행

- 트레이 메뉴에서 "자동 실행" 토글
- 레지스트리 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`에 등록/해제
- 설정 저장: `%APPDATA%\TextSniperWin\config.json`

### 첫 실행

- 최초 실행 시 Balloon Tip 표시: "TextSniper가 실행 중입니다. Shift+Alt+T를 눌러 캡처를 시작하세요."
- 한국어 OCR 언어팩 미설치 감지 시 안내 메시지 표시 (영어 전용 모드로 동작 가능)

## 에러 처리

| 시나리오 | 처리 방법 |
|----------|-----------|
| OCR 언어팩 미설치 | 첫 실행 시 안내 + 영어 전용 폴백 |
| 글로벌 단축키 등록 실패 | 트레이 툴팁에 표시, 트레이 메뉴로 수동 캡처 가능 |
| 선택 영역 10x10px 미만 | 캡처 취소 처리 |
| OCR 결과 빈 문자열 | 클립보드 쓰지 않음 + 실패 아이콘 표시 |
| 클립보드 쓰기 실패 | 실패 아이콘 표시 |
| 보호된 창/관리자 앱 캡처 (검은 화면) | Known Limitation으로 문서화 |
| 앱 중복 실행 | 기존 인스턴스 포커스, 신규 인스턴스 종료 |
| OCR 처리 5초 초과 | 타임아웃 후 실패 아이콘 표시 |

## Known Limitations

- `BitBlt` 기반 캡처이므로 관리자 권한으로 실행된 앱(UAC 상승 프로세스)의 화면은 캡처 불가 (검은 화면)
- `SetWindowDisplayAffinity`로 보호된 창(DRM 콘텐츠 등)은 캡처 불가
- 전체화면 DirectX 독점 모드 앱에서는 오버레이 표시 불가
- 멀티 모니터 환경에서는 마우스 커서가 있는 모니터만 캡처 가능 (v2에서 전체 모니터 지원 예정)

## 제외 사항 (MVP 범위 외)

- 히스토리(기록) 저장
- 클라우드 OCR 연동
- 한국어/영어 외 다국어 지원
- 이미지 파일 저장
- 인식 결과 편집 (인라인 에디터)
- 복사 전 미리보기 팝업
- 줄바꿈 자동 정리 옵션
- 키보드만으로 영역 선택 (접근성)
- 멀티 모니터 전체 커버 오버레이
- 단축키 커스터마이징
